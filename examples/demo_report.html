Invention Summary

The manuscript discloses Dense Object Nets, a system comprising:

  • A fully-convolutional neural network (34-layer stride-8 ResNet) that maps full-resolution RGB images to dense descriptor spaces providing a D-dimensional descriptor vector per pixel
  • A self-supervised training procedure using pixelwise contrastive loss with pixel matches/non-matches generated automatically from 3D RGBD reconstructions
  • Object masking via 3D reconstruction-based change detection to focus learning on objects of interest
  • Background domain randomization using autonomously acquired object masks to enforce cross-scene descriptor consistency
  • Hard-negative scaling that adaptively normalizes the non-match loss by the number of active hard negatives rather than total non-matches
  • Multi-object distinct descriptor learning via three strategies: cross-object loss, direct training on multi-object scenes, and synthetic multi-object scenes via mask layering
  • Selective class generalization or instance specificity achieved by modifying the loss function and sampling procedure
  • Application to robotic manipulation: grasping specific points on deformed objects by finding best descriptor matches via L2 nearest-neighbor lookup, with class-general and instance-specific variants.

While distilled feature fields from large vision models have enabled semantic correspondences across 3D scenes, their features are point-based and restricted to object surfaces, limiting their capability of modeling complex semantic feature distributions for hand-object interactions. Being able to reliably locate the same anatomical structure across varying images is a fundamental task in medical image analysis. In principle it is possible to use landmark detection or semantic segmentation for this task, but to work well these require large numbers of labeled data for each anatomical structure and sub-structure of interest. A coarse-to-fine strategy ensures both global and local anatomical information are encoded. Negative sample selection strategies are designed to enhance the embedding's discriminability. Using SAM, one can label any point of interest on a template image and then locate the same body part in other images by simple nearest neighbor searching. We demonstrate the effectiveness of SAM in multiple tasks with 2D and 3D image modalities. On a chest CT dataset with 19 landmarks, SAM outperforms widely-used registration algorithms while only taking 0.23 seconds for inference. On two X-ray datasets, SAM, with only one labeled template image, surpasses supervised methods trained on 50 labeled images. We also apply SAM on whole-body follow-up lesion matching in CT and obtain an accuracy of 91%. SAM can also be applied for improving image registration and initializing CNN weights.
